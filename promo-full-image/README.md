# Promo Full Image

https://mitchellveix.github.io/drupal-components/promo-full-image/

A standalone recreation of a Drupal promo full image component originally built for the Visit Sarasota website.

## Folder structure

```text
promo-full-image/
├── index.html
├── css/
│   └── promo-full-image.css
├── scss/
│   └── promo-full-image.scss
├── images/
│   ├── promo-full-image1.jpg
│   ├── promo-full-image2.jpg
│   ├── promo-full-image3.jpg
└── README.md
```

## Notes

- Drupal-specific wrappers, field classes, lazy-loading attributes, and entity metadata were removed.
- Internal Drupal paths were converted to full Poverty Point URLs.
- The original empty heading was given visible demo text.
- The second image received descriptive alternative text for the standalone demo.
- Both the original SCSS structure and compiled standalone CSS are included.

## Usage

Open `index.html` directly in a browser or serve the folder through GitHub Pages.

## Image assets

The included raster images are labeled placeholders because the source site blocked automated asset downloads in this environment. Replace them with the original files using the same filenames, and no HTML or CSS changes will be needed.
